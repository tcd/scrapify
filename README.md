# Falsification

![Shopify API Version](https://img.shields.io/badge/Shopify_API-2019--10-brightgreen.svg)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'falsification'
```

And then execute:

```sh
bundle
```
Or install it yourself as:

```sh
gem install falsification
```
